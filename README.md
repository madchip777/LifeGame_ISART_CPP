## What/Why

This is my repo of a C++ "Jeu de la vie" (Game's of life), to serve as a codding skill proof for Isart school candidature.

## Run 

Linux use :
    
    `./GameOfLife`

Windows use :

    `./GameOfLife.exe`

## Build

To **compile** this project, as **root**, in the **project root**, run :

    `g++ -o GameOfLife main.cpp`
